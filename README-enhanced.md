# ShakaQuest - 中学受験社会学習アプリ（拡張版）

中学受験の社会科（地理・歴史・公民）をDuoLingoライクなゲーミフィケーション要素で楽しく学習できるWebアプリケーションです。

## 🌟 新機能（拡張版）

### 📚 全科目対応
- **地理**: 47都道府県、気候、産業、工業地帯、伝統工業、公害問題
- **歴史**: 旧石器時代～昭和時代まで17時代区分、重要人物・事件
- **公民**: 日本国憲法、三権分立、基本的人権、地方自治、国際関係

### 🎮 高度なゲーミフィケーション
- **レベルシステム**: 11段階のレベル（学習開始～社会科グランドマスター）
- **バッジシステム**: 12種類の実績バッジ（初回挑戦、各科目マスター、完璧主義者など）
- **経験値システム**: 難易度別XP（基本10XP、標準15XP、応用20XP）+ 時間ボーナス
- **コインシステム**: 正解数 × 10コイン
- **連続学習ストリーク**: 毎日学習で継続記録

### 📊 詳細な学習分析
- 科目別正答率・学習進捗
- カテゴリ別成績分析
- 今日の学習目標とその達成度
- 時間別学習効率分析

### 🎯 多様な問題形式
- **4択問題**: 基本的な知識確認
- **穴埋め問題**: 重要用語の記憶定着
- **マッチング問題**: 関連付け学習
- **地図選択問題**: 地理的理解の確認

## 🚀 技術仕様

## 🎓 学習コンテンツ詳細

### 地理分野
- **都道府県**: 47都道府県と県庁所在地（18箇所が異なる）
- **気候**: 6つの気候区分（北海道、日本海側、太平洋側、内陸性、瀬戸内、南西諸島）
- **農業**: 専業農家・兼業農家、穀倉地帯、主要農産物
- **水産業**: 三大漁港、漁業の種類
- **工業**: 三大工業地帯（京浜・中京・阪神）、工業地域
- **伝統工業**: 各地の伝統工芸品
- **公害**: 四大公害病、環境問題

### 歴史分野
- **17時代区分**: 旧石器時代～昭和時代
- **重要人物**: 各時代の重要人物と功績
- **政治制度**: 時代ごとの政治体制の変化
- **文化史**: 各時代の文化的特徴
- **年号・事件**: 重要な出来事と年代

### 公民分野
- **憲法**: 日本国憲法三大原則（国民主権・基本的人権・平和主義）
- **政治**: 三権分立（立法・行政・司法）
- **人権**: 基本的人権の種類と内容
- **地方自治**: 住民自治と団体自治
- **国際関係**: 国際連合と国際協力
- **経済**: 財政・税制・社会保障

## 💡 使用方法

### 基本的な学習フロー
1. **科目選択**: 地理・歴史・公民から選択
2. **カテゴリ選択**: 学習したい分野を選択（全カテゴリも可能）
3. **クイズ実行**: 時間制限付きで問題に回答
4. **即座の解説**: 正誤に関わらず詳細な解説を表示
5. **結果分析**: 正答率、獲得XP、学習時間の詳細分析

### 問題形式の特徴
- **時間制限**: 1問30秒（集中力向上）
- **即座のフィードバック**: 回答後すぐに正解・解説表示
- **視覚的な理解**: カラフルなUI、アイコン使用
- **段階的難易度**: 基本→標準→応用の3段階
